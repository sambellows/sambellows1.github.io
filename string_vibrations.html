Vibrations of Strings

In 1D, the eigenfunctions are

$\Psi_n(x) = Acos(k_nx) + Bsin(k_n x)$

For Dirichlet boundary conditions at $x = 0$ and $x = L$, $A = 0$, $\Psi_n(L) = 0$ and $k_n = n\pi/L$. These eigenfunctions are used, for example, to describe the vibrations of a string fixed at both ends.

Eigenfunctions for a vibrating string fixed at both ends.


For Neumann boundary conditions at $x = 0$ and $x = L$, $B = 0$ and $k_n = n\pi/L$. These eigenfunctions are used, for example, to describe the vibrations of a string free at both ends.

Eigenfunctions for a vibrating string free at both ends.


For a Dirichlet boundary condition at $x = 0$ and a Neumann boundary conditions at $x = L$, $B = 0$ and $k_n = n\pi/2L$, with $n = 1, 3, 5, ...$. These eigenfunctions are used, for example, to describe the vibrations of a string fixed at one end and free at the other.

Eigenfunctions for a vibrating string fixed at one end and free at the other.
